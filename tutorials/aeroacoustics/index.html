Aeroacoustic Noise Analysis

FLOWUnsteady uses PSU-WOPWOP (a FW-H code) and a BPM code for calculating tonal and broadband aeroacoustic noise, respectively. PSU-WOPWOP is not included in the FLOWUnsteady suite, but rather the user can request a binary of PSU-WOPWOP directly from the developer at Penn State University.

The example under examples/rotornoise/singlerotor.ipynb shows instructions and validation of the aeroacoustics solver. Further validation of rotor-on-rotor interactions is given in the following paper by the authors: Alvarez, E. J., Schenk, A., Critchfield, T., and Ning, A., “Rotor-on-Rotor Aeroacoustic Interactions of Multirotor in Hover,” Journal of the American Helicopter Society, Jul. 2020, (in review).
